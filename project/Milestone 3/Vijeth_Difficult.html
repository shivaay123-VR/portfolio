How the Internet Works

DNS

DNS or Domain Name System Converts a website name (google.com) into its IP address.DNS is the phonebook of internet.Humans access information online through domain names,like google.com or espn.com.Web browsers interact through Internet Protocol (IP) addresses. DNS translates domain names to IP addresses so browsers can load internet resources.

Is a company that gives you internet.Its an organisation that conects us to myraids of services related to accessing managing using or participating in the internet.Internet services provided by ISP's include internet across,internet transit,domain name registration,web hosting and colocation.ISPs can be organised in various forms,such as commercial comunity owned,non profit,or otherwise privately owned.

Client

The device making the request.Defination of client in computing would be a computer that gets information from another computer called server in the context of client server model of computer networks.THe server is often on another computer system,in which case the client accesses the service by the way of a network.It is an operation,relies on sending a request to another program or a computer hardware or software that accesses a service made available by the server
