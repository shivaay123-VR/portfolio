HTML

HTML stands for HyperText Markup Language. It’s the language used to make web pages. Every webpage you see has HTML running behind it

Html-

is one of the most useful languages present.It forms the skeleton of every website

Html the word in itself gives a lot of information about itself

  • The word breaks down into the following
  1. HyperText: “Hyper” means more than normal. Hypertext is text that can link to other pages or documents (like clicking a blue link).
  2. Markup: Markup means adding symbols or tags to text so the browser knows how to display it.
